## 📖프로젝트 개요

KT AIVLE School 4기 미니프로젝트 7차 : 수화 번역기 웹 사이트<br>
-> 수어를 알파벳으로 번역해주는 AI 기반 웹 서비스<br><br>
**사용 기술 :** Django, MLFLOW, ChatGPT API, AWS, Anaconda 




<br>

## 🧾프로젝트 수행 내용
- Django를 사용하여 웹 프로젝트를 진행하였습니다.
- MLFlow를 사용하여 모델 서버를 사용하였습니다.
- AWS를 이용하여 서버를 구축하였습니다.
